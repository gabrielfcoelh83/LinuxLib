// como funciona

Três passos. Zero fricção.

01

Pesquise ou navegue

Use nome do comando, função ou palavra-chave. A busca entende contexto, não só sintaxe exata.

$ buscar "redirecionar porta"
→ ssh -L, iptables -t nat, socat...
02

Leia a explicação

Descrição clara, parâmetros essenciais explicados um a um e um exemplo prático direto ao ponto.

-L local_port:host:remote_port
# redireciona porta local para remota
03

Aplique e compartilhe

Copie e execute no terminal agora. Encontrou uma variação útil? Contribua com a comunidade.

$ ssh -L 8080:localhost:80 user@host
✓ Copiado para área de transferência
