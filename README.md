# 🐧 LinuxLib — Domine o terminal Linux

**LinuxLib** é uma biblioteca open‑source de comandos Linux com uma interface moderna, busca dinâmica e experiência otimizada para produtividade.  
Ela foi projetada para ajudar profissionais de TI, estudantes e entusiastas a **encontrar, entender e aplicar comandos Linux** sem perder tempo com buscas dispersas ou documentação excessivamente técnica.

## ✨ Funcionalidades

- 🔍 **Busca em tempo real** com filtro por nome, descrição ou categoria.
- ⌨️ **Navegação por teclado** (`↑`, `↓`, `Enter`, `Esc`, `Ctrl+K` / `Cmd+K`).
- 📋 **Cópia rápida** do comando – clique no card inteiro ou no botão "Copiar".
- 🧠 **Modal de detalhes** com explicação aprofundada e exemplo prático.
- 📱 **Responsivo** – funciona perfeitamente no desktop e mobile.
- ♿ **Acessível** – tags ARIA, foco visível, suporte a teclado e leitores de tela.
- 🎨 **Visual premium** – efeito glow, gradientes, animações suaves e tema escuro nativo.
- ⚡ **Sem dependências externas** – apenas HTML, CSS e JavaScript puro.
